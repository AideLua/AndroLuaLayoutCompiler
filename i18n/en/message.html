How to use

  1. Fill in the information into the edit box above
  2. Generate a PATH
  3. Click "Export LayoutHelper" to export "LayoutHelper.lua" to the "Project/lua" directory. AideLua is "Project/Module/Type/src/luaLibs").
  4. Click the "Compile Layout" button and select the save path.

Notice

  1. Must import `android.widget.AdapterView`, otherwise the inheritance relationship cannot be determined.
  2. Must fill in the PATH, otherwise the sublayout file cannot be found.
  3. This program is not adapted to SAF framework because it needs to load files dynamically, so please make sure to give storage permission, otherwise the software will not work.
