使用方法

  1. 填入信息到上面的编辑框内
  2. 生成 PATH
  3. 点击“导出 LayoutHelper”，将 “LayoutHelper.lua” 导出到“项目/lua”目录内（AideLua为“项目/模块/类型/src/luaLibs”）。
  4. 点击“编译布局”按钮，选择保存路径。

注意事项

  1. 必须导入 `android.widget.AdapterView`，否则无法判断继承关系。
  2. 必须填写 PATH，否则无法找到子布局文件。
  3. 本程序因为需要动态加载文件，未适配 SAF 框架，所以请务必给予存储权限，否则软件将无法工作。
